我推荐

A / 单主链画布

优点

  • 最接近 Dify / Coze 的节点工作流感
  • 最适合固定流程但允许拖动排布
  • 后续加节点、加缩放、加小地图都最顺

注意

  • 需要做真正的 pan / zoom / drag，不是普通页面滚动
  • 节点详情要用右侧面板或弹窗，不然会太挤
